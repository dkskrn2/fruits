# B2B 도매시장 조사

## 1. 한국 농산물 도매시장 현황

### 기존 도매 유통 구조
```
산지 농가 → 산지 유통인 → 도매시장 (경매) → 중도매인 → 소매상 → 소비자
```
- 유통 단계가 많아 **마진 누적** → 최종 가격 상승
- 도매시장 경매 중심 거래 → **가격 변동성** 큼
- 농가 수취율 낮음 (소비자가 대비 30~40% 수준)

### 도매시장의 디지털 전환
- **KAFB2B** (농수산물 온라인도매시장): aT 운영, 2023년 말 정식 출범
  - 판매 수수료 0%
  - 구매대금 30일 무이자 여신
  - 전국 단위 시공간 제약 없는 거래
- 기존 도매시장의 한계 극복을 위한 **온라인 거래 플랫폼** 확대 추세

## 2. B2B 과일 도매 고객 특성

### 주요 고객 유형
| 고객 유형 | 특성 | 니즈 |
|----------|------|------|
| **대형 소매체인** | 대량 정기 구매, 품질 기준 엄격 | 안정적 공급, 품질 일관성, 가격 경쟁력 |
| **식당/레스토랑** | 지속적 소량~중량 구매 | 신선도, 배송 신속성, 다양한 품목 |
| **호텔/케이터링** | 프리미엄 과일, 대량 주문 | 고급 품질, 맞춤 포장, 정기 납품 |
| **식품 가공업체** | 가공용 과일 대량 구매 | 저렴한 가격, 대량 공급, 규격 통일 |
| **온라인 소매업체** | 위탁판매/재판매 | 상품 DB, 드롭쉬핑, 포장·배송 대행 |
| **과일 전문점** | 프리미엄/희소 품종 소량 구매 | 품종 다양성, 희소성, 정보 제공 |
| **학교/기관** | 단체급식용 대량 구매 | 안전 인증, 가격 경쟁력, 정기 납품 |

### B2B 구매 의사결정 요인 (우선순위)
1. **가격 경쟁력** (볼륨 디스카운트)
2. **품질 일관성** (등급, 당도, 크기 균일)
3. **공급 안정성** (물량 확보, 정기 납품)
4. **결제 조건** (여신, 외상, 세금계산서)
5. **배송 신뢰성** (콜드체인, 시간 정확도)

## 3. B2B 플랫폼 핵심 기능 요구사항

### 거래 기능
- 대량 주문 시스템 (MOQ 설정)
- 볼륨 할인 자동 적용
- 견적 요청/응답 시스템
- 계약 기반 정기 거래
- 거래처별 맞춤 가격표

### 결제/정산
- 세금계산서 자동 발행
- 후불 결제 (여신 시스템)
- 월 단위 정산
- 선결제 할인

### 물류
- B2B 전용 배송 (팔레트, 박스 단위)
- 지정 시간 납품
- 전국 콜드체인 네트워크
- 반품/교환 프로세스

### 정보 제공
- 시세 정보 (도매가 실시간)
- 입고 예정 물량 공유
- 품질 인증서/검사 결과
- 산지 이력 추적

## 4. B2B 시장 진입 전략
1. **초기**: 소규모 과일 전문점·온라인 재판매업체 대상
2. **확장**: 식당/레스토랑 → 호텔/케이터링
3. **성숙**: 대형 소매체인, 기관 납품
4. **차별화**: 산지직결 + 데이터 기반 가격 투명성
