# B2C 소비자 행동 조사

## 1. 소비자 프로필

### 주요 타겟 세그먼트

| 세그먼트 | 연령대 | 특성 | 구매 패턴 |
|---------|--------|------|----------|
| **건강 지향 MZ세대** | 25~39세 | 건강식, 비건, 유기농 관심 | 모바일 주문, SNS 영향, 구독 선호 |
| **맞벌이 가정** | 30~45세 | 시간 부족, 편의성 중시 | 새벽배송, 대량 주문, 정기배송 |
| **1인 가구** | 20~40세 | 소포장, 간편 가공 선호 | 소량 다품종, 즉시 배송 |
| **프리미엄 소비자** | 35~55세 | 고품질, 브랜드 가치 중시 | 선물세트, 고당도 과일, 희소 품종 |
| **가성비 추구형** | 전 연령 | 가격 민감, 비교 구매 | 대량 묶음, 비등급 상품, 프로모션 활용 |
| **시니어** | 55세+ | 건강 관심, 디지털 전환 중 | 전화 주문 병행, 자녀 대리 주문 |

## 2. 구매 행동 분석

### 온라인 과일 구매 동기
1. **편의성** (47%): 집에서 주문, 배송까지 원스톱
2. **가격 비교** (28%): 여러 플랫폼 비교 후 최저가 구매
3. **품질 정보** (15%): 당도, 원산지, 재배 방식 확인 가능
4. **시간 절약** (10%): 마트 방문 불필요

### 구매 시 중요 고려 요인
1. **신선도/품질** — 가장 중요 (리뷰/사진 확인)
2. **가격** — 가성비 또는 프리미엄 가치
3. **배송 속도** — 새벽/당일 배송 선호
4. **원산지** — 국산 제철 과일 선호
5. **포장 상태** — 파손 방지, 보냉 포장

### 재구매 결정 요인
- 배송 품질 (파손 없이 신선하게 도착)
- 맛 일관성 (기대한 당도/식감)
- CS 대응 (불량 시 즉시 교환/환불)
- 멤버십/할인 혜택

## 3. 소비 채널별 특성

| 채널 | 비중 (추정) | 특성 |
|------|------------|------|
| **모바일 앱** | 65% | 주력 채널, 간편결제, 푸시 알림 |
| **PC 웹** | 15% | 대량 주문, 상세 정보 확인 |
| **SNS 커머스** | 10% | 인스타/유튜브 기반 구매, 충동구매 |
| **라이브커머스** | 7% | 실시간 소통, 현장감, 할인 |
| **전화/기타** | 3% | 시니어 고객, 대량 주문 |

## 4. 트렌드별 상품 기회

### 구독 서비스
- **제철 과일 박스** (월 1~2회 배송)
- **이달의 과일** 큐레이션 (신품종 소개)
- **건강 과일 세트** (항산화, 비타민 특화)
- 구독자 비중: 온라인 식료품 구매자 중 37%

### 선물 시장
- 명절 선물세트 수요 지속 성장
- **프리미엄 과일 선물** (고급 포장, 보증서 첨부)
- 기업 선물 대량 주문 (B2B2C 연계)

### 간편 과일
- 컷팅 과일 세트
- 세척 후 바로 먹는 과일
- 과일+요거트/시리얼 조합 세트

## 5. B2C 핵심 성공 요인
1. **첫 구매 경험**이 재구매를 결정 → 첫 주문 품질 관리 최우선
2. **비주얼 마케팅**: 과일 사진/영상 품질이 구매 전환율 직결
3. **리뷰 시스템**: 실구매 리뷰가 신뢰도 핵심
4. **CS 대응 속도**: 불량 과일 즉시 교환/환불 → 브랜드 신뢰
5. **개인화**: 구매 이력 기반 추천으로 재구매율 향상
