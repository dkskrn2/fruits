## 2. 경쟁사 상세 분석

### 마켓컬리 (Kurly)
**강점**
- 프리미엄 신선식품 큐레이션 강점
- 2024년 첫 EBITDA 흑자 달성 → 재무 건전성 개선
- 네이버 협업으로 고객층 확대 예정
- 뷰티, 패션 등 비식품 확장 '투 트랙' 전략

**약점**
- 가격대가 높은 편 → B2B/도매 고객 확보 어려움
- 프리미엄 포지셔닝으로 가성비 고객 흡수 제한적

### 오아시스마켓
**강점**
- 13년 연속 흑자 → 안정적 수익 모델 검증
- '착한 가격의 건강한 식탁' → 가성비 + 건강 동시 충족
- 유기농/무첨가/친환경 식품 전문

**약점**
- 배송 지역 제한적 (수도권 중심, 충청권 확대 중)
- 티몬 인수 부담 가능성

### 쿠팡 (로켓프레시/산지직송)
**강점**
- 압도적 물류 인프라 (전국 새벽배송)
- 산지 직접 매입 → 유통 중간 마진 제거
- 2025년 과일 매입량 29% 증가
- 가격 경쟁력 최강

**약점**
- 비전문적 큐레이션 (대량·범용 상품 중심)
- 개별 농가 특성을 살린 차별화 어려움

## 4. 시장 포지셔닝 기회

```
   프리미엄
      ↑
   컬리  │  [기회: B2C 프리미엄 과일 전문]
         │
 ────────┼──────── → 과일 전문성
         │
 오아시스 │  쿠팡
   가성비 │
      ↓
```

### 차별화 포인트 (Blue Ocean)
1. **과일 전문 플랫폼**: 기존 종합 신선식품 플랫폼과 차별화
2. **B2B + B2C 듀얼 모델**: 도매업체와 일반고객을 동시 서비스
3. **산지-소비자 직결**: 중간유통 최소화
4. **데이터 기반 큐레이션**: 당도·산도·품종별 맞춤 추천
5. **제철 과일 구독**: 정기배송 서비스 특화
