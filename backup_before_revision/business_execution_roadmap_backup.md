**2. CAC & LTV 분석**

**CAC (Customer Acquisition Cost) 목표**:

```
월 마케팅비: 300만원
목표 신규 고객: 120명
→ CAC = 300만원 ÷ 120명 = 25,000원/명
```

**채널별 CAC**:
- 네이버 검색: 20,000원 (전환율 2%)
- 네이버 쇼핑: 20,000원
- 인스타그램: 40,000원 (인지도 목적, 직접 전환 낮음)
- 블로그: 30,000원 (간접 전환)

**LTV (Lifetime Value) 계산**:

```
선물형 고객:
- 1차 구매: 89,000원 (샤인머스캣)
- 재구매율: 25% (명절 반복)
- 2차 구매: 75,000원 (사과·배 세트)
- LTV = 89,000 + (75,000 × 0.25) = 107,750원

일상형 고객:
- 1차 구매: 35,000원 (가족 사과 5kg)
- 재구매율: 40% (월 1회 반복)
- 연간 구매: 35,000 × 5회 = 175,000원
- LTV = 175,000원 (1년 기준)

가중평균 LTV:
- 선물형 70%: 107,750 × 0.7 = 75,425원
- 일상형 30%: 175,000 × 0.3 = 52,500원
- → 평균 LTV = 127,925원
```

**LTV/CAC Ratio**:
```
127,925원 ÷ 25,000원 = 5.1배
→ 양호 (3배 이상이면 건강한 비즈니스)
```
