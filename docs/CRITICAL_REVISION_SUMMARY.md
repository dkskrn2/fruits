## 🔴 치명적 문제 Top 5

### 1. 판매할수록 손실 발생
```
현재 가격(89,000원):
- 실제 원가: 97,500원
- 매 판매 손실: -8,500원
- 100개 판매 = 850만원 손실
```
**해결: 최소 95,000원, 이상적으로 105,000원**

### 2. 4개월 내 파산
```
현재 자본(75.55M원):
- Month 4 예상 잔액: 0원
- 실제 필요: 197.9M원
```
**해결: 최소 150M원 확보 필수**

### 3. 창업자 과로사 위험
```
Month 4부터:
- 예상 근무: 주 80시간
- Month 6: 주 100시간+
```
**해결: Month 3까지 운영 매니저 필수 채용**

### 4. 여름철 평판 파괴
```
6-8월 부패율:
- 딸기: 40%
- 예상 클레임: 25-40%
```
**해결: 딸기 판매 중단 or 콜드체인 투자**

### 5. 인플루언서 의존 함정
```
현재 계획:
- 인플루언서 70% 의존
- 최대 용량: 200-300 주문/월
- 목표 500 주문: 달성 불가
```
**해결: 채널 다각화, 인플루언서 40%로 축소**
