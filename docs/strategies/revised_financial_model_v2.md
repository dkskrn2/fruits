## 2. 고객 생애 가치 (LTV) 분석

### 2.1 선물 고객 세그먼트 (전체 고객의 70%)

```
첫 구매: 95,000원 (샤인머스캣)
재구매율: 15% (실제 데이터 기반, 계절성 고려)
2차 구매: 85,000원 × 0.15 = 12,750원
3차 구매: 75,000원 × 0.05 = 3,750원

연간 총 매출: 111,500원
연간 구매 횟수: 1.2회
평균 객단가: 92,917원

기여이익 기준 LTV:
- Year 1: 2,901 + (2,901 × 0.15) = 3,336원
- Year 2: 3,336 × 0.3 = 1,001원
- 2년 LTV: 4,337원

CAC 대비: 4,337 / 25,000 = 0.17x ❌ 치명적
```
