## 3. 가격 인상 정당화 전략

### 3.1 고객 커뮤니케이션

#### A. 가격 ≠ 비싸다, 가치 = 합리적이다

```markdown
온브릭스 대비 +5,000원이 비싼가요?

[온브릭스]
- 샤인머스캣 2kg: 94,900원
- 당도 보증: 있음
- 실패 시: 부분 환불

[우리]
- 샤인머스캣 2kg: 95,000원 (+100원)
- 당도 보증: 있음
- 실패 시: 100% 재발송
- 추가: 전문가 큐레이션 서비스

결론: 100원 더 내고 확실한 보증
```

#### B. 큐레이션 수수료 개념 도입

```markdown
일반 과일 가격: 85,000원
+ 전문가 선별 수수료: 5,000원
+ 100% 품질 보증: 3,000원
+ 즉시 재발송 보험: 2,000원
= 큐레이션 가격: 95,000원

"과일값이 아닙니다.
 실패 없는 선택을 사는 것입니다."
```

## 5. 동적 가격 정책 (Dynamic Pricing)

### 5.1 재고 기반 가격 조정

```python
def calculate_price(base_price, stock_level, days_until_expiry):
    """
    기본가격, 재고수준, 유통기한으로 가격 결정
    """
    if days_until_expiry <= 2:
        # 긴급 처분
        return base_price * 0.5
    elif days_until_expiry <= 3:
        # 할인 판매
        return base_price * 0.7
    elif stock_level > 150:
        # 재고 과다
        return base_price * 0.9
    elif stock_level < 20:
        # 재고 부족 (프리미엄)
        return base_price * 1.1
    else:
        # 정상
        return base_price

# 예시:
# 월요일: 재고 200개, D-5 → 85,500원 (10% 할인)
# 수요일: 재고 50개, D-3 → 95,000원 (정상가)
# 금요일: 재고 15개, D-1 → 47,500원 (50% 할인)
```

### 5.2 수요 예측 기반 가격

```markdown
AI 기반 수요 예측:
- 날씨 (비오는 날 -20% 수요)
- 계절 (여름 샤인머스캣 +30% 수요)
- 이벤트 (월급날 +15% 수요)
- 경쟁사 (쿠팡 할인 시 -10% 수요)

자동 가격 조정:
- 수요 110% 예상 → 가격 +5%
- 수요 90% 예상 → 가격 -5%
- 수요 70% 예상 → 프로모션 발동
```
